# LIMA WordPress Theme

One-page WordPress theme for LIMA - Limpieza de Metales

## Installation Instructions

1. **Zip the theme folder:**
   - Put all theme files in a folder named `lima`
   - Zip the entire `lima` folder

2. **Upload to WordPress:**
   - Go to WordPress Admin Dashboard
   - Navigate to Appearance → Themes
   - Click "Add New" → "Upload Theme"
   - Choose your zipped theme file
   - Click "Install Now"
   - Click "Activate"

## Theme Setup

### 1. Upload Your Logo
- Go to Appearance → Customize → Site Identity
- Upload your LIMA logo (use the color version: LIMA_color.png)
- Save changes

### 2. Set Hero Background Image
- Go to Appearance → Customize → Hero Section
- Upload your hero background image (hero-goff.jpg or similar)
- Customize the hero text if needed
- Save changes

### 3. Create Navigation Menu
- Go to Appearance → Menus
- Create a new menu called "Primary Menu"
- Add custom links for:
  - Nosotros (#nosotros)
  - Servicios (#servicios)
  - Galería (#galeria)
  - Contacto (#contacto)
- Assign to "Primary Menu" location
- Save menu

### 4. Add Contact Information
- Go to Appearance → Customize → Contact Information
- Add your email, phone, and address
- Save changes

### 5. Optional: Install Contact Form 7
- Install Contact Form 7 plugin for the contact form
- Or use the built-in fallback form

## Files Included

- `style.css` - Main stylesheet with theme header
- `index.php` - Main template file
- `functions.php` - Theme functions and setup
- `header.php` - Header template
- `footer.php` - Footer template
- `front-page.php` - Home page template
- `js/navigation.js` - Navigation functionality
- `js/smooth-scroll.js` - Smooth scrolling

## Customization

All theme colors and settings can be customized through:
1. WordPress Customizer (Appearance → Customize)
2. Editing `style.css` for design changes
3. Editing template files for structure changes
